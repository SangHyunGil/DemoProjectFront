### Installing

```markdown
npm install 혹은 yarn install
```

### Executing program

local에서 돌리기 위해서는 spring 백엔드 코드가 필요합니다. 
백엔드 repo 주소는 다음과 같습니다 (https://github.com/SangHyunGil/KonerBackEnd.git, 현재 private)
이후
```
npm run start or yarn start
```
